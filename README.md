# #52 Case Converter

A hyper-minimalist text formatting tool for instant case transformation. 1 Tool / 1 Action.

## Core Philosophy
- **1 Tool / 1 Action**: Focuses solely on switching text cases (UPPER, lower, Capitalize, Sentence).
- **Performance**: Lightweight, browser-only, and optimized for both mobile and low-spec PCs.
- **Privacy**: No login, no tracking, no ads. Your text never leaves your browser.

## Features
- **UPPERCASE**: Converts all characters to capital letters.
- **lowercase**: Converts all characters to small letters.
- **Capitalize**: Capitalizes the first letter of every word.
- **Sentence**: Capitalizes only the first letter of the first word (sentence case).
- **One-Tap Copy**: Quick "Copy to Clipboard" button for seamless workflow.

## Design System
- **Background**: #ffffff
- **Main Color**: #333333
- **Accent**: #eeeeee (Input area: #fafafa)
- **Border Radius**: 15px (Strictly applied to all elements)
- **UI Style**: Tool-like, bold sans-serif interface (iOS/Android native standard).

## Usage
1. Paste your text into the input area.
2. Select the desired case format button.
3. Tap "Copy to Clipboard" to use the result.
